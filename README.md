# Drum Scores

Interactive drum sheet music in your browser.

Each song is a LilyPond-engraved score paired with a playable drum kit:
press **Play** and a cursor follows the music note-by-note while real drum
samples (CC0 *Virtuosity Drums*) play in sync. Slow the tempo down to 5%
for hard fills, or sync to a YouTube backing track to play along with the
band.

Tested on iPad — built for music-school practice rooms.

**Live site:** https://schneider128k.github.io/drum_scores/
